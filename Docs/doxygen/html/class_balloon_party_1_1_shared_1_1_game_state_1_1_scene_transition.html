Detailed Description

General-purpose scene transition component.

Attach to any GameObject and wire Load to a Button's onClick event in the Inspector. When _preload is enabled the target scene is loaded additively on Start with rendering disabled so all game logic (DI, pool pre-warming) runs in the background. Load restores rendering and unloads the current scene.
