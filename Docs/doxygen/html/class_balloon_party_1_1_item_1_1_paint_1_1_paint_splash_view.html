Detailed Description

Poolable paint-splash effect.

Extends EffectView so it participates in the standard effect-pool pipeline via SimplePoolChannel<TItem>. The prefab holds pre-placed ColorableRenderer children (one per potential neighbor — 6 for a hex grid) with the PaintBlob shader material. Splash particles are spawned as independent pooled instances via ParticlePoolChannel so they outlive the view's pool return. Call ISplashEffect.PrepareDisplay with target data before Play.
