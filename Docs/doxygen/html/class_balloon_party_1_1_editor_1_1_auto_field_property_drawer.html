Detailed Description

Base class for property drawers that want common fields drawn automatically via serialized-property reflection, with a manually-handled section for fields that need special layout (variable height, conditional visibility, etc.).

Subclasses must override ExcludedFields, GetSpecialFieldsHeight, and DrawSpecialFields. Optionally override BuildFoldoutLabel, DrawPinnedFields, and GetPinnedFieldsHeight for fields that must appear above the auto-drawn common section.
