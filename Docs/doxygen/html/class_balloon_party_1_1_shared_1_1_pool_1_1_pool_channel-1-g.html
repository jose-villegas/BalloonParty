TItem BalloonParty.Shared.Pool.PoolChannel< TItem >.Peek ( )

Returns the most recently returned item without removing it from the pool or activating it.

Useful for reading state that was preserved on the component before it was returned (e.g. rotation snapshots). Returns null if no items are available.
