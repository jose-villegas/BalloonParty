◆ SetupFollow()

float duration,
Action onArrived,
bool useUnscaledTime = false,
TrailMotion motion = TrailMotion::Default )

Homes on a live-updating target rather than a fixed point: over duration it eases from its launch position to targetProvider ()'s current value along the move curve for motion , firing onArrived at the end.

Lets a trail chase a moving object (an overflow balloon still sliding as the pile compacts) and land on it exactly, with the same curve control as the fixed-point flights.
