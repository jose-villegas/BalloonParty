Detailed Description

Poolable chain-lightning effect.

Extends EffectView so it participates in the standard effect-pool pipeline via SimplePoolChannel<TItem>. Visual parameters are serialized on the prefab. Call PrepareDisplay with target data before Play.

Color tint,

Starts the chain-lightning animation fire-and-forget.

Call PrepareDisplay first. position and tint are unused — positions are set via PrepareDisplay and the line material is not tinted. onComplete is invoked after the full retraction.

Implements BalloonParty.Shared.IEffect.

Index 0 must be the item-balloon world position; subsequent entries are same-color balloon positions sorted nearest-first. onTargetHit is invoked per jump (index 0 = first target).
