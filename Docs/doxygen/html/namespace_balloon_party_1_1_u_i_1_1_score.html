enum BalloonParty.UI.Score.TrailMotion

Selects which move/scale curve pair a FlyingTrail flies with.

The trail holds a collection indexed by this enum (O(1) lookup); a value with no curve set falls back to the trail's default pair, so adding a motion is opt-in per prefab and per call site.
