# Projectile

The ball the player fires at the balloon grid.

A projectile travels in a straight line, bounces off the side walls, and pops any balloon it collides with. It can carry shields that absorb hits before the projectile itself is consumed. Once it comes to rest or is destroyed, a new one is loaded into the thrower.

The projectile interacts with the thrower (which launches it), the grid (which it hits), and the balloon (which it destroys on contact).
